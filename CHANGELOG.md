# 更新日誌 (Changelog)

所有重要變更都會記錄在此檔案中。

## [1.2] - 2025-12-18

### 新增功能
- 實現動態共享記憶體機制，每個 RDP 連線實例有獨立的共享記憶體名稱
- 添加鍵盤鎖定狀態同步功能（NumLock、CapsLock、ScrollLock）
- 支援滑鼠中鍵和滾輪操作
- 實現防手震（debounce）機制，避免滑鼠快速點擊時的誤觸問題
- 在 RdpBridge 中添加 rdpb_get_shm_name 和 rdpb_sync_locks 函數
- 支援每個連線實例獨立的共享記憶體空間，便於多連線同時運行

### 技術改進
- 改進共享記憶體管理，使用動態名稱避免多連線衝突
- 優化滑鼠事件處理，支援更多滑鼠按鍵和滾輪操作
- 實現鍵盤鎖定狀態自動同步，確保本機與遠端狀態一致
- 改進 RdpBackend 類別，支援動態共享記憶體大小調整
- 在視窗焦點獲取時自動同步鍵盤鎖定狀態

### 修復問題
- 解決多連線時共享記憶體衝突問題
- 改善滑鼠點擊的準確性，減少誤觸情況
- 修復 NumLock 等鎖定鍵狀態不同步的問題

## [1.1] - 2025-12-15

### 新增功能
- 使用 PySide6 和 QOpenGLWidget 實現 GPU 加速渲染
- 實現 memoryview 零複製技術，提升影像傳輸效能
- 添加暴力輪詢計時器，確保即時響應
- 改進滑鼠事件處理，支援遊戲視角轉動順暢性
- 添加強制 OpenGL 硬體加速設定

### 技術改進
- 使用 PySide6 替代原有 UI 框架
- 實現 GPU 加速影像渲染
- 零複製記憶體管理，避免不必要的記憶體複製
- 提升影像串流效能和流暢度
- 改進連線心跳機制，確保穩定連線

### 修復問題
- 解決影像顯示效能問題
- 改善記憶體管理，避免影像複製開銷
- 優化滑鼠事件處理，確保遊戲操作流暢

## [1.0] - 2024-12-15

### 新增功能
- 實現基本 RDP 連線功能
- 添加即時桌面串流顯示
- 實現滑鼠控制功能（點擊、移動、滾輪）
- 實現鍵盤控制功能（支援修飾鍵和功能鍵）
- 添加游標顯示功能
- 實現視窗焦點管理機制
- 添加多執行緒架構以確保 UI 流暢性

### 技術改進
- 使用共享記憶體 (SHM) 進行高效能影像傳輸
- 實現完整的鍵盤掃描碼對應表
- 添加視窗關閉事件處理
- 實現妥善的記憶體管理機制

### 修復問題
- 解決按鍵卡死問題（確保按鍵釋放事件正確發送）
- 改善視窗焦點檢測機制
- 優化影像串流效能
