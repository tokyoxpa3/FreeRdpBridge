# 更新日誌 (Changelog)

所有重要變更都會記錄在此檔案中。

## [1.1] - 2025-12-15

### 新增功能
- 使用 PySide6 和 QOpenGLWidget 實現 GPU 加速渲染
- 實現 memoryview 零複製技術，提升影像傳輸效能
- 添加暴力輪詢計時器，確保即時響應
- 改進滑鼠事件處理，支援遊戲視角轉動順暢性
- 添加強制 OpenGL 硬體加速設定

### 技術改進
- 使用 PySide6 替代原有 UI 框架
- 實現 GPU 加速影像渲染
- 零複製記憶體管理，避免不必要的記憶體複製
- 提升影像串流效能和流暢度
- 改進連線心跳機制，確保穩定連線

### 修復問題
- 解決影像顯示效能問題
- 改善記憶體管理，避免影像複製開銷
- 優化滑鼠事件處理，確保遊戲操作流暢

## [1.0] - 2024-12-15

### 新增功能
- 實現基本 RDP 連線功能
- 添加即時桌面串流顯示
- 實現滑鼠控制功能（點擊、移動、滾輪）
- 實現鍵盤控制功能（支援修飾鍵和功能鍵）
- 添加游標顯示功能
- 實現視窗焦點管理機制
- 添加多執行緒架構以確保 UI 流暢性

### 技術改進
- 使用共享記憶體 (SHM) 進行高效能影像傳輸
- 實現完整的鍵盤掃描碼對應表
- 添加視窗關閉事件處理
- 實現妥善的記憶體管理機制

### 修復問題
- 解決按鍵卡死問題（確保按鍵釋放事件正確發送）
- 改善視窗焦點檢測機制
- 優化影像串流效能

## [0.9.0] - 2025-12-15

### 新增功能
- 初始 RDP 連線功能
- 基本影像串流顯示
- 簡單滑鼠操作支援
- 鍵盤輸入轉發功能

### 注意事項
- 此為初始測試版本
- 可能存在效能和穩定性問題
